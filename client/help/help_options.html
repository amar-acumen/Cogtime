Options

7. Options

If you changed your options settings, and later you want to restore to the original state, you can click the Restore button. When you click the Restore button, it will pop up a alert window "Clear current cache and restore?". After clicking "YES" button, the current tab cache will be cleared and restored to the default system settings.

(1) Message
Chat user can make personal setting on private message acceptance, i.e., choose to accept PM from members only or friends only.

(2) Sound
Chat user may make personal setting to play or mute any kind of sound to himself/herself.

(3) Cache
Chat user can set and manage the local cache here.

(4) Theme
Users have options to choose from various chat themes and there are altogether 10 themes available: classic, bubble, point line, plain rect, serene, gradient rect, orkut, gradient bubble, clover and sunset glow.

Single Line: Display the message in the same line with the username, or start a new line.

Big Avatar: Display big, small or no avatar at all.

(5) Video Layout
The “Video Layout” tab won’t show in Options unless your admin has enabled the video module already.
Video open mode and dock position can be set here. More details

(6) Password
Registered users can change the password in Options->Password.
