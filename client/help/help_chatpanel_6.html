11) Add Smilies

Chat users may add some smilies to conversations to express themselves better either by clicking an icon from the smiley panel or by typing some special keyboard shortcut if they can remember them. This function is available in the private chat window as well as in the chat lobby.

12) Send a Flash Emotion

Flash emotions are animations to play in the chat to show special emotion, like happy or angry, etc. This function is available in the private chat window as well as in the chat lobby. A flash emotion won't automatically play unless a user has configured his own setting to do so.
