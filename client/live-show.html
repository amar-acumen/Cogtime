Introduction

It’s a special read-only chat client, which can be used to broadcast the live chat content in a certain room of the standard chat client to attract more users to join in, or spread site news, sports scores, stock price, etc.

The Live Show is a pure HTML client, and it must be used together with the Flash chat client, because the previous one is a reader, and the latter one is the content provider.
