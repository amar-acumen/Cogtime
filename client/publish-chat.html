How to add the chat to website?

The 123FlashChat server is successfully installed and started, but just locally, now it's time to launch the chat room online to your website.

Brief instructions:

  • Step 1. Publish the chat client folder
    Copy <123 flashchat installed dir/client> folder to your web server document root directory.
  • Step 2. Generate Code
    Enter the following admin panel-> Code Generator, choose a particular client type, style, settings and then generate code.
